## Predicción del Nivel de PIB utilizando datos del Banco Mundial

Este proyecto tiene como objetivo predecir el nivel de PIB de distintos países utilizando indicadores económicos, sociales y demográficos obtenidos desde el Banco Mundial.

El trabajo se desarrolla en tres etapas principales:
- Etapa 1: Análisis descriptivo e imputación de datos
- Etapa 2: Reducción de dimensionalidad con PCA
- Etapa 3: Modelación mediante algoritmos de clasificación
